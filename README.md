# atlas2 -- the atlas, rebuilt from scratch

Everything here is generated by `python3 build.py` from:

  - `data/walks/` -- dent-walk atlases (bends per dent set, from the
    bendprover walker; certified records are the source of truth)
  - `builders/`   -- salvaged converters (glb.py from the old atlas)

`build.py` is rerunnable at any time and regenerates `site/` whole.
Nothing in `site/` is hand-edited. The old atlas at ~/Dropbox/neo/atlas
stays untouched until this replaces it.

Modules so far:
  dentings -- every non-pancake prime net v<=13 with all of its
              dentings mod symmetry, canonical order, undented first.
